# YogaFlow Website

## About The Project

YogaFlow is a modern and responsive yoga and wellness website developed using React.js.  
The project is designed to provide a smooth and user-friendly experience with clean UI components, responsive layouts, and easy navigation.

The website includes multiple sections such as services, blog pages, authentication pages, and user interaction components focused on wellness and yoga activities.
